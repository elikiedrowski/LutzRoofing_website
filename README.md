# Lutz Roofing Website

A modern, professional website for Lutz Roofing Co., Inc. - Michigan's leading commercial roofing contractor.

## Features

- **Modern Design**: Clean, professional aesthetic inspired by contemporary web design patterns
- **Responsive Layout**: Fully responsive design that works on all devices
- **Service Pages**: Dedicated pages for all roofing systems (EPDM, TPO/PVC, Built-Up, Modified, Metal, Green Roofs)
- **AI Chat Widget**: Interactive chat widget for customer engagement
- **Safety Focus**: Dedicated page highlighting Lutz Roofing's commitment to workplace safety
- **Warranty Information**: Comprehensive warranty manufacturer information
- **Contact Forms**: Easy-to-use contact forms for estimate requests

## Design Philosophy

This website combines:
- The modern, clean design aesthetic from The CRM Wizards
- All original content from lutzroofing.com
- Professional roofing industry presentation
- Easy navigation and user experience

## Color Scheme

- **Primary Blue**: #1e3a8a (navy blue for professionalism)
- **Accent Orange**: #ea580c (energetic accent color)
- **Dark**: #0f172a (text and headers)
- **Light Grays**: For backgrounds and subtle elements

## Setup

1. Clone or download this repository
2. Open `index.html` in a web browser
3. For development, use a local server (e.g., Live Server in VS Code)

## Customization

### Adding Images
Replace placeholder images in the `assets/images/` directory:
- `hero-roofing.jpg` - Homepage hero image
- Service-specific icons in `assets/images/icons/`
- Partner logos in `assets/images/partners/`

### Updating Content
- Edit HTML files directly for content changes
- Modify `assets/css/styles.css` for styling adjustments
- Update `assets/js/main.js` for functionality changes

## AI Chat Widget

The AI chat widget is currently set up with simulated responses. To integrate with a real AI service:

1. Modify the `handleMessage()` function in `main.js`
2. Add API integration for your chosen AI service
3. Update responses to provide real-time assistance
